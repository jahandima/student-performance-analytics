# Student Performance Analytics

## Project Overview
This project explores factors affecting student academic success using Exploratory Data Analysis (EDA) and progression modeling. The goal is to identify patterns in score gaps and predict how students progress over time.

## Dataset
The analysis is based on student performance data including:
- **Demographics:** Age, gender, family size.
- **Social Factors:** Study time, free time, alcohol consumption, and health.
- **Academic:** Previous grades and school support.

## Key Analysis
1. **Score Gap Analysis:** Comparing performance across different subjects (Math vs. Portuguese).
2. **Progression Modeling:** (Currently in progress) Modeling how scores change from G1 to G3.

## How to Run
1. Clone the repo: `git clone https://github.com/jahandima/student-performance-analytics.git`
2. Activate virtual environment: `source .venv/Scripts/activate`
3. Open `stdt_prfm_dataset/mat_score_gap_analysis.ipynb` in Cursor or Jupyter.
